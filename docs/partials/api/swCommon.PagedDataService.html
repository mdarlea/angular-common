PagedDataService
service in module swCommon

Description

Service that queries paged data . It can be used with ngGrid, any grind that supports paging or with an infinit-scrolling list

Dependencies

  • $http
  • $q
  • ngAuthSettings

Usage

PagedDataService(baseUrl, data);

Parameters

ParamTypeDetails
baseUrlstring
  • base URL for calling the API
dataObject

Data paging settings

PropertyTypeDetails
fixedPageboolean

True for paged grids, False for infinite scrolling lists

loadingboolean

True is data is currently loading, False otherwise

itemsArray

The items to be displayed

totalRecordsnumber

Total number of loaded records

selectedArray

Selected records

totalPagesnumber

Total pages loaded

filterOptionsObject

Options for filtering data

filterTextstring Text to filter data

sortOptionsObject

Options for sorting data

fieldsArray Sort data by the given fields
directionsArray 'asc' for sorting arcending, 'desc' for sorting descending

pagingOptionsObject

Options for paging

pagingOptions.pageSizesArray

A list of possible page sizes. This option aplies only to paged grids

pagingOptions.pageSizenumber

Specifies the number of items on a single page

pagingOptions.currentPagenumber

The current page. This option aplies only to paged grids
